# Sales & Revenue Analysis Dashboard

## Project Overview
This project is an interactive Sales & Revenue Analysis Dashboard developed using Power BI. It provides insights into sales performance, revenue trends, customer segments, and product performance.

## Tools Used
- Power BI
- Power Query
- DAX
- CSV Dataset

## Dashboard Features
- Total Sales
- Total Profit
- Total Orders
- Total Quantity
- Monthly Sales Trend
- Sales by Category
- Sales by Region
- Top 10 Products
- Customer Segment Analysis
- Interactive Slicers

## Dashboard Pages
1. Executive Dashboard
2. Sales Analysis
3. Profit Analysis
4. Customer Insights
